# Misfar – Backend API 🚀

AI-Powered Smart Tourism Platform for Oman

## About
This is the backend server for Misfar, an AI-driven tourism platform for Oman.
Built with Node.js, Express.js, and PostgreSQL.

## Tech Stack
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** PostgreSQL (Neon.tech)
- **Authentication:** JWT
- **AI Integration:** Claude API (LLM chatbot)
- **Cloud:** AWS (EC2, IAM, S3)

## Features
- 🔐 JWT Authentication & Authorization
- 👤 User management & role-based access control
- 🗺️ Destinations, activities, and alerts API
- 🤖 LLM chatbot integration
- 🛠️ Admin Dashboard API
- 💾 PostgreSQL database with Neon.tech
